# Data Visualization in Python — Case Study

## Overview  
This case study explores app store data using Python to uncover trends, patterns, and relationships in app characteristics such as ratings, installs, genres, Android versions, and content categories. The notebook demonstrates how data visualization turns raw data into meaningful insights.

---

## Tools & Libraries Used

- Python  
- Pandas — Data cleaning and analysis  
- Matplotlib — Plotting  
- Seaborn — Statistical visualization  
- NumPy — Numerical operations  

---

## Dataset Description

The dataset includes information about mobile applications:

| Column | Description |
|--------|-------------|
| App | Application name |
| Category | App category |
| Rating | User rating |
| Reviews | Number of reviews |
| Size | App size |
| Installs | Number of installs |
| Type | Free or Paid |
| Price | App price |
| Content Rating | Target audience age group |
| Genres | App genre |
| Last Updated | Last update date |
| Android Ver | Required Android version |

---

## Key Analysis Performed

- Distribution of app ratings  
- Relationship between app size and ratings  
- Ratings comparison across top genres  
- Popular Android versions  
- Minimum and average ratings by content category  
- Trends by update month  
- Bar plots, box plots, and joint plots  

---

## Visualizations Included

- Bar plots  
- Box plots  
- Joint plots  
- Distribution plots  
- Category comparisons  

These help identify performance trends, rating distributions, and app characteristics impact.
